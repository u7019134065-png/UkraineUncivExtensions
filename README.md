# UkraineUncivExtensions

This repository contains a minimal test mod for Unciv.

## Test Mod

A simple test mod located in the `TestMod/` directory that adds:
- **Test Nation** - a custom civilization with a unique ability
- **Test Warrior** - a unique unit unlocked by Test Technology
- **Test Technology** - an early Ancient era tech

To use the mod, copy the `TestMod` folder to your Unciv mods directory.
